Navigation Test

Testing the navigation between landing page and create account page.

Go to Landing Page Go to Create Account

Test Steps:

  1. Click "Go to Landing Page"
  2. Click "Register Now" button
  3. Click "Back to Home" link
  4. Check if you return to the same landing page design

Expected Result:

You should see the same dark-themed landing page with the Prism effect both times.

If you see a different design:

There might be a caching issue or a different page being loaded. Try:

  • Hard refresh (Ctrl+F5)
  • Clear browser cache
  • Check browser console for errors
