Navigation Test

This page tests the navigation between landing page and create account page.

Test Navigation Links:

Go to Landing Page Go to Create Account Page

Expected Behavior:

  1. Click "Go to Landing Page" - should show the Ezana Finance landing page
  2. Click "Register Now" button on landing page - should go to create account page
  3. Click "Back to Home" on create account page - should return to landing page

Current Issue:

If you're seeing a "completely differently designed website" when clicking "Back to Home", there might be:

  • A caching issue in your browser
  • A different landing page being loaded
  • A CSS loading issue
  • A JavaScript redirect happening
