Navigation Component Demo

This page demonstrates the navigation component that matches the sidebar structure. The navigation includes:

  • Dashboard - Direct link to home dashboard
  • Research Tools - Dropdown with Inside the Capitol, Market Analysis, Company Research, Economic Indicators, and Financial Analytics
  • Watchlist - Direct link to watchlist page
  • Community - Direct link to community page
  • User Menu - Profile dropdown with settings and logout options

Features

The navigation component provides:

  • Responsive design that adapts to mobile, tablet, and desktop
  • Dropdown menus with hover and click activation
  • Mobile-friendly collapsible menu
  • Keyboard navigation support
  • Active state highlighting
  • User profile management
  • Easy configuration through JavaScript

Usage

To use this navigation component in your pages:

  1. Include the CSS and JavaScript files
  2. The navigation will be automatically generated
  3. Customize through the configuration file
  4. Use the JavaScript API for dynamic control
